Javascript 变量作用域

  • 全局作用域
  • 函数作用域

在 js 中只有 2 种变量作用域，变量的查询会形成一条链式结构，这就形成了作用域链，全局作用域可以在任何地方访问到，然而函数作用域只能在函数内部访问到，外层无法访问；

上面代码console.log(a)会进行变量查询，它会优先在fn()函数内部查询，如果存在同名变量，那么直接返回，停止查询。如果内部没有回继续向外部查询,fn 函数内部不存在会向全局查找，全局存在则返回，不存在返回 undefined

上面代码先从内部查询变量a，因为 fn 中存在 a 所有直接返回，查询结束；
上面 2 种情形都是无法访问到 fn 内部的变量 b 的。

闭包

闭包作为一个链接函数内部与外部的桥梁，可以使得函数外部访问到函数内部的变量，由于变量被外部引用，所以变量会一直存在于内存之中，闭包使用不当会导致内存溢出，程序奔溃。
