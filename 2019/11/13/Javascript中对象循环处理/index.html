Javascript 对象循环的几种方式

  • for in循环
  • Object.keys()
  • Object.getOwnPropertyNames()
  • Object.entries()

for in：

  • 遍历对象自身可枚举属性键名

Object.keys()：

  • 返回对象自身可枚举属性键名集合

Object.getOwnPropertyNames()：

  • 返回对象自身所有属性健名集合包括可枚举与不可枚举
