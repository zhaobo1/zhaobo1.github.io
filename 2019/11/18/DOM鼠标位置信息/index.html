鼠标对象（mouseEvent）位置信息

- clientX,clientY
- offsetX,offsetY
- pageX,pageY
- screenX,screenY

（一）clientX,clientY

事件被触发时鼠标指针相对于`浏览器页面（或客服区）的水平/垂直坐标`。缩放浏览器不会受到影响。
{% asset_img clientXY.jpg clientX,clientY %}

（二）offsetX,offsetY

事件被触发是鼠标指针相对于`事件源对象的水平/垂直坐标`。缩放浏览器不会受到影响。
{% asset_img offsetXY.jpg offsetX,offsetY %}

（三）pageX,pageY

事件被触发时鼠标指针相对于`文档（document）的水平/垂直坐标`。缩放浏览器不会受到影响。
 {% asset_img pageXY.jpg pageX,pageY %}

（四）screenX,screenY

事件被触发时鼠标指针相对于`电脑屏幕的水平/垂直坐标`。缩放浏览器不会受到影响。

（五）scrollX,scrollY

滚动条滚动的水平垂直距离。
pageX = clientX + scrollX;
pageY = clientY + scrollY;
